## Timing Constants

| Token | Value | Derivation |
|-------|-------|------------|
| `--t-phi` | 382ms | 1000 / phi squared |
| `--t-phi-half` | 191ms | 382 / 2 |
| `--ease-phi` | cubic-bezier(0.382, 0, 0.618, 1) | Control points at 1/phi squared and 1/phi |
| `--ease-out` | cubic-bezier(0.16, 1, 0.3, 1) | Standard decelerate curve |
| `--ease-move` | cubic-bezier(0.22, 1, 0.36, 1) | Layout motion |
| `--t-fast` | 99ms | phi-scaled base timing |
| `--t-base` | 160ms | Base duration |
| `--t-slow` | 419ms | phi * base |
| `--t-drift` | 678ms | phi squared * base |
